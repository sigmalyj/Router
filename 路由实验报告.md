# 计算机网络实验二——路由实验报告

## 运行程序

Ubuntu: 

1. 第一个终端当中运行pox

```bash
cd /home/ubuntu/Router/code && conda activate router && python /opt/pox/pox.py --verbose ucla_cs118
```

如果显示6633端口占用，则先进行清理

```bash
sudo lsof -i :6633
sudo kill -9 <PID>
```

2. 第二个终端当中运行Mininet

```bash
cd /home/ubuntu/Router/code && conda activate router && chmod +x run.py && sudo ./run.py
```

成功，可直接执行

3. 第三个终端当中构建并运行Router

```bash
cd /home/ubuntu/Router/code && conda activate router && make
```
